<!-- Run Locally -->
### :running: Run Locally

Clonar el proyecto

```bash
  git clone https://github.com/Anng3l/DAWEB-Prueba-1.git
```

Instalar las dependencias

```bash
  npm i
```

Correr el servidor

```bash
  npm run dev
```
